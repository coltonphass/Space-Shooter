# \# 🚀 Space Shooter

# 

# A classic top-down space shooter built with Python and Pygame. Navigate through space, dodge meteors, blast enemies, and survive as long as you can in this fast-paced arcade experience!

# 

# \## 🎮 Demo

# 

# !\[Space Shooter Demo](./assets/demo.gif)

# 

# \## ✨ Features

# 

# \- \*\*🎯 Fast-Paced Gameplay\*\* - Move your ship around the screen and shoot meteors with responsive controls

# \- \*\*📊 Score Tracking\*\* - Survive longer and destroy more enemies to rack up a higher score

# \- \*\*💥 Animated Explosions\*\* - Dynamic visual feedback with satisfying explosion effects when meteors are destroyed

# \- \*\*🔊 Immersive Audio\*\* - Includes laser sound effects, explosion sounds, and atmospheric background music

# \- \*\*⚡ Ready to Play\*\* - Executable included for instant gameplay without Python installation

# 

# \## 🎯 How to Play

# 

# \### Option 1: Run the Executable (Recommended)

# 1\. \*\*Download the game:\*\*

# &nbsp;  ```bash

# &nbsp;  git clone https://github.com/coltonphass/Space-Shooter.git

# &nbsp;  cd space-shooter

# &nbsp;  ```

# 

# 2\. \*\*Launch the game:\*\*

# &nbsp;  - Double-click `main.exe` (Windows)

# &nbsp;  - No Python installation required!

# 

# \### Option 2: Run from Source

# 1\. \*\*Prerequisites:\*\*

# &nbsp;  - Python 3.13 or higher

# &nbsp;  - Pygame-CE 2.5.5

# 

# 2\. \*\*Install dependencies:\*\*

# &nbsp;  ```bash

# &nbsp;  pip install pygame-ce==2.5.5

# &nbsp;  ```

# 

# 3\. \*\*Run the game:\*\*

# &nbsp;  ```bash

# &nbsp;  python main.py

# &nbsp;  ```

# 

# \## 🎮 Controls

# 

# | Action | Keys |

# |--------|------|

# | Move Ship | `Arrow Keys` or `WASD` |

# | Shoot Lasers | `Space` |

# | Quit Game | `ESC` or close window |

#
